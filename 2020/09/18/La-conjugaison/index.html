• 过去分词性数配合规则
être + PP 无 COD 与主语性数配合 Elle est rentrée très tard cette nuit.
avoir + PP (+ COD) 无 COD / COD 正序 不性数配合 Elle a parlé de venir nous voir.
Elle a fait son choix en cinq minutes.
se + être + PP（绝对意义、被动意义） 无 COD 与主语性数配合 La réunion s’est bien passée.（绝对意义）
Ces livres se sont vite vendus.（被动意义）
se + être + PP（自反意义、相互意义） se 作 COD 与 COD 性数配合
即与主语性数配合		 Elle s’est lavée.
se + être + PP（自反意义、相互意义） se 作 COI 不性数配合 Elle s’est lavé les cheveux.
COD + … + être/avoir + PP COD 前置 与 COD 性数配合 Cette femme, dans sa jeunesse, il l’a aimée à la folie
C’est une de mes amies que je t’ai présentée.
C’est une de mes amies que je t’ai présentées.
COD + … + être/avoir + 无人称动词 PP COD 前置，作形式宾语，实际主语 不性数配合 J’imagine l’énergie qu’il nous a fallu pour faire ce travail.
en/le 作 COD + … + être/avoir + PP COD 前置 不性数配合 J’ai trouvé des pommes fraîches et j’en ai acheté.（副代词en）
- Vous savez que le prof est malade ? - Oui, on nous l’a dit.（中性代词le）
